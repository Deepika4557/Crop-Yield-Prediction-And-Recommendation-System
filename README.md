This project focuses on analyzing Canadian agricultural data to understand the key factors influencing crop yield. It applies exploratory data analysis techniques to identify patterns, trends, and relationships across variables such as climate, soil conditions, and farming practices. In addition, multiple machine learning models are developed to predict crop yield accurately. The project also includes a recommendation system that suggests suitable crops based on environmental and farm-specific conditions. Overall, this work demonstrates how data-driven approaches can support better decision making in agriculture.
